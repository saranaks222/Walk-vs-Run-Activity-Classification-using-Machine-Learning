# Walk vs Run Activity Classification using Machine Learning

## Project Overview

Human Activity Recognition (HAR) is an important application of Machine Learning that helps identify physical activities performed by individuals using sensor-generated data.

This project focuses on classifying human activities into two categories: Walking and Running. Machine Learning algorithms are used to analyze sensor data and accurately predict the activity being performed.

---

## Problem Statement

Activity recognition plays a significant role in healthcare, fitness tracking, sports analytics, and wearable technology. Manual monitoring of activities can be difficult and time-consuming.

The objective of this project is to develop a machine learning model that can automatically classify whether a person is walking or running based on sensor measurements.

---

## Objectives

- Classify activities as Walk or Run.
- Analyze sensor-based activity data.
- Build a supervised machine learning model.
- Evaluate model performance using classification metrics.
- Demonstrate practical applications of Human Activity Recognition.
